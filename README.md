# DeviConHan - 恶魔链接汉化项目

原文链接：[原作者 Guideline](https://bayachao.com/devil-connection/guideline.html)

## 📥 安装说明

1. 确保你已经拥有原版游戏（且能正常运行）。
2. 下载本项目的最新 [Releases](../../releases) 版本。
3. 将解压后的文件复制到游戏根目录（即包含 `DevilConnection.exe` 的文件夹）。
4. 运行汉化补丁程序。
